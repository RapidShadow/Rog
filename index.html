About

Once not so long ago, Zack was Juuling in the bathroom. He thought if foods should be organic why should vaping be any different? In 2018, RØG was born in San Francisco, using only 100% real tabacco.

We think if food should be organic, vaping should be organic too. RØG use state of the art design to invent a single use vape made by nature. Unlike traditional vaping systems using electricity, our system requires fires like our primal ancestors. Every pack of RØG comes with 20 single use vape sticks. Each stick use non GMO tabacco with non artifical flavors.
